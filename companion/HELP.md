## SCS

This module controls SCS by Show Cue Systems.

### Configuration
* Type in the device IP address and port.

### Available actions
* Play (cue)
* Stop (cue)
* Pause / Resume (cue)
* Goto (cue)
* Activate Hotkey
* Activate Note on Hotkey
* Deactivate Hotkey
* Set level on Master fader
* Go
* Pause / Resume all
* Stop
* Stop all
* Fade all
* Goto Top of cuelist
* Goto End of cuelist
* Goto Previous cue in playlist
* Goto Next cue in playlist
